Для установки выполните следующие шаги:
1. Для корректной работы используйте MySQL-5.7
2. Импортируйте базу данных из указанного пути /Инструкция/a25.sql
3. В файле /app/db.php необходимо указать доступ к базе данных
	пример:
		$user = "root"; // имя пользователя
    		$pass = "qwerty123"; // пароль
    		$host = "localhost"; // хост
    		$db = "a25"; // имя базы данных
